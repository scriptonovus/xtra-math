- **ADMIN CONTROL**: Full access to upload, edit, and delete files
- **READ-ONLY ACCESS**: Public users can view files without modification rights
- **MARKDOWN SUPPORT**: Automatic rendering of .md files with full formatting
- **NEON INTERFACE**: Hacker-themed cyberpunk aesthetic with glowing green terminal style
- **SECURE AUTHENTICATION**: Password-protected admin access
